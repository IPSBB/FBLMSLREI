POS Credit Limit
================
Set warning limit and blocking credit limit to the customers and warn or block them while using the Customer Account payment method in the
POS payments based on their previous credits.

Configuration
=============
* After Installation, go to Invoicing - Configuration-> Settings -> Enable 'Sales Credit Limit' and set limits.
