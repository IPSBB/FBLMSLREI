Overview

This Odoo module tracks product price changes. It provides a list view and graphical representation of the price change history. If you need any help, please contact abdullah@bisolv.com.

This new action allows viewing the price change history for the specific product.
Graphical representation of price changes by date.
List view of price changes. Records are limited to a maximum of 100; exceeding this limit will be automatically cleared.
